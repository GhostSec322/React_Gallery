# [React -Gallery 프로젝트]

- 이 프로젝트는 Pixabay API와 Firebase를 결합하여 사용자가 무료 이미지를 검색하고 다운로드하는 기능과 갤러리를 관리하는 기능을 제공하는 웹 애플리케이션제작 프로젝트 입니다.

  - 사용기술
    - React
    - Redux
  - API
    - Pixabay api (이미지 다운로드 ,검색)
    - Firebase( 이미지 업로드 카테고리 설정 DB연동 ,사용자 인증)
